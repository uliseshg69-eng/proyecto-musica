# 🎵 Proyecto Música

Una aplicación web moderna para reproducir música, gestionar playlists y disfrutar de tu música favorita.

## ✨ Características

- 🎵 **Reproductor de Música** - Controla la reproducción con play, pausa, siguiente y anterior
- 📊 **Visualizador** - Animaciones visuales sincronizadas con la música
- 🎧 **Control de Volumen** - Ajusta el volumen de la música
- ⏱️ **Barra de Progreso** - Visualiza el progreso de la canción actual
- 📝 **Información de Canción** - Artista y nombre de la canción
- 🎨 **Interfaz Moderna** - Diseño responsive y atractivo
- 📱 **Mobile Friendly** - Compatible con dispositivos móviles

## 🚀 Instalación

1. Clona el repositorio:
```bash
git clone https://github.com/uliseshg69-eng/proyecto-musica.git
cd proyecto-musica
```

2. Abre `index.html` en tu navegador web

## 📁 Estructura del Proyecto

```
proyecto-musica/
├── index.html          # Página principal
├── styles.css          # Estilos CSS
├── script.js           # Lógica JavaScript
└── README.md           # Este archivo
```

## 🎮 Cómo Usar

1. **Reproducir música**: Haz clic en el botón Play (▶️)
2. **Pausa**: Haz clic en el botón Pausa (⏸️)
3. **Siguiente canción**: Haz clic en el botón Siguiente (⏭️)
4. **Canción anterior**: Haz clic en el botón Anterior (⏮️)
5. **Ajusta el volumen**: Usa el control deslizante de volumen
6. **Salta a un momento**: Haz clic en la barra de progreso

## ⌨️ Atajos de Teclado

- **Espacio**: Play/Pausa
- **Flecha Derecha**: Siguiente canción
- **Flecha Izquierda**: Canción anterior

## 🛠️ Tecnologías Utilizadas

- **HTML5** - Estructura
- **CSS3** - Estilos y animaciones
- **JavaScript** - Funcionalidad interactiva
- **Web Audio API** - Procesamiento de audio
